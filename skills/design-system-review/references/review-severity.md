# 设计审查分级

## P0

页面破版、核心流程不可用、主要内容重叠或移动端无法完成关键操作。

v0.1 脚本不做完整视觉截图判断，P0 主要预留给截图审查或后续 Playwright 集成。

## P1

明显违反视觉系统，需要阻断：

- 出现未登记品牌色、主操作色或大面积背景色。
- 按钮、卡片等核心组件出现明显圆角漂移。
- 页面使用与 `DESIGN.md` 完全冲突的视觉基调。

## P2

局部不一致：

- 非核心区域出现未登记字号。
- 局部 spacing 或 radius 与规范不一致。
- 个别组件没有复用抽象组件规则。

## P3

可优化：

- 命名不统一。
- 可读性或层级可进一步增强。
- 需要补充规范但不影响当前页面使用。
